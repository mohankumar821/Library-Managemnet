# 📚 LibraryHub - Modern Library Management System

A **beautiful, responsive front-end dashboard** for library management built with **pure HTML, CSS, and vanilla JavaScript**. 

## ✨ **Features**
- **Glassmorphism Design** - Modern UI with backdrop blur effects
- **Fully Responsive** - Works perfectly on desktop, tablet & mobile
- **Interactive Elements** - Hover animations, focus states, and smooth transitions
- **Dashboard Stats** - Real-time metrics for books, members & transactions
- **Quick Actions** - One-click access to common library operations
- **Advanced Search** - Unified search across books, members & transactions
- **Navigation Tabs** - Easy section switching (Dashboard, Books, Members, Reports)
